📖 วิธีใช้งาน
1
ตั้งค่า API → ไปแท็บ ตั้งค่า → กรอก Universe ID และ API Key → กด บันทึก
2
ดู Ban List → ไปแท็บ รายการ Ban → กด รีเฟรช → เห็นรายชื่อทั้งหมด
3
Ban ผู้เล่น → ไปแท็บ Ban → กรอก UserId + ชื่อ + เหตุผล → กด Ban
ban คนออฟไลน์ได้เลย ไม่ต้องให้เขาออนไลน์
4
Unban ผู้เล่น → ไปแท็บ Unban → กรอก UserId → กด Unban
หรือกดปุ่ม ✅ Unban ในรายการ Ban โดยตรง
🎮 วิธีใช้ใน Roblox (In-game):
กด M เพื่อเปิด Admin Panel → กดปุ่ม Ban → เลือก player หรือพิมพ์ UserId ใน Unban box (สีทอง) → กด Unban
🏗️ วิธี Deploy เว็บนี้
1
สร้าง GitHub repo ใหม่ → อัปโหลดไฟล์ทั้งหมด (index.html, netlify.toml, netlify/functions/ds.js)
2
ไปที่ netlify.com → Add new site → Import from GitHub → เลือก repo
3
Netlify จะ deploy อัตโนมัติ ใช้เวลา ~1 นาที → ได้ URL เช่น yourname.netlify.app
4
เปิด URL → กรอก API Key → ใช้งานได้เลย ✅ ไม่ต้องรัน proxy ใดๆ
